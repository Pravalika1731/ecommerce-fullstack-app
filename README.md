# 🛒 E-Commerce Full Stack App

## 🚀 Features
- Add / Update / Delete Products
- Image Upload & Display
- Cart Functionality
- Search Products

## 🛠 Tech Stack
- React.js
- Spring Boot
- REST API
- MySQL / H2

## ▶️ Run Project

### Backend
cd Ecommerce-Backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm run dev
